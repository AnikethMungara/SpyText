# Example Documents

This directory will contain test documents for SpyText analysis.

## Phase 1

Currently empty - no documents needed yet.

## Phase 2+

Will contain:
- Sample PDF documents (text-based and scanned)
- Test cases with various visibility issues
- Adversarial examples for testing

## Creating Test Documents

Examples to create in future phases:

1. **white-on-white.pdf** - White text on white background
2. **microscopic.pdf** - Text at 0.5pt font size
3. **off-screen.pdf** - Text positioned outside page bounds
4. **low-contrast.pdf** - Dark gray text on black background
5. **legitimate.pdf** - Normal, readable document
6. **mixed.pdf** - Mix of visible and invisible content
7. **prompt-injection.pdf** - Hidden "ignore previous instructions" text
